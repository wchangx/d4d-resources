Typography Cheat Sheet

Typeface

A typeface is a set of characters that share stylistic unity. It is comprised of an alphabet of letters, numbers, punctuation and diacritical marks.

Font

A complete character set of a particular typeface in one size.
Also, an electronic file that renders a typeface in all sizes.

Line-height

Stay between 1.3em and 1.6em for legibility.
